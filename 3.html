Object-oriented programming language developed by Sun Microsystems in 1995. The founder is James Gosling. When Oracle acquired Sun Microsystems in 2010, it owned the copyright of Java. Currently, Oracle JDK, which is distributed by OpenJDK by GPL2 or Oracle, has been strengthening its pay-as-you-go policy since January 2019. Java EE is owned by the Eclipse Foundation. Java languages have been leading the development of Java Community Process (JCP) since J2SE 1.4.
